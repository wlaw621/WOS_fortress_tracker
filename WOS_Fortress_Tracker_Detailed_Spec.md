# WOS 요새쟁탈 명단작성 PRO 시스템 상세서

본 문서는 **1953 GOM 연맹**의 요새전 운영 효율화를 위해 개발된 명단작성 도구의 기술 명세 및 향후 업데이트 가이드를 포함합니다.

---

## 1. 시스템 개요
- **서비스 명**: WOS 요새쟁탈 명단작성 PRO (Fortress Tracker Pro)
- **주요 목적**: 요새전 참여 스크린샷 기반 명단 추출, 등급 기반 자동 정렬, 실시간 마스터 데이터 관리
- **개발 환경**: React (Tailwind CSS, Lucide Icons, Gemini AI API, Google Apps Script)

---

## 2. 핵심 기능 상세

### 2.1 AI 명단 추출 (Gemini 1.5 Flash)
- **입력**: 스크린샷 이미지 (JPG/PNG)
- **처리**: 이미지 내 텍스트에서 닉네임만 정제하여 JSON 데이터로 변환
- **특이사항**: `runAI` 함수를 통해 여러 장의 이미지를 한 번에 처리 가능

### 2.2 지능형 데이터 매칭 및 정렬
- **대조 로직**: 추출된 닉네임과 구글 시트 기반 `masterList`를 대조 (부분 일치 포함)
- **컬럼 구성**: 이름, 등급(R5~R1), 분류(운영진/본캐/부캐) 3개 데이터 관리
- **등급 할당**: 마스터 시트에 등록된 정보를 우선 적용하며, 미등록 유저는 자동으로 '본캐/R3' 등급 부여
- **정렬 순서**:
    1. 분류 우선순위 (운영진 > 본캐 > 부캐)
    2. 등급 순 (R5 > R4 > R3...)
    3. 가나다(사전) 순

### 2.3 세션 기반 관리 (12시/18시/21시)
- 상단 탭을 통해 시간대별 독립적인 명단 데이터 관리
- 각 시간대별로 '초기화' 기능 제공

### 2.4 인앱 마스터 리스트 관리 (Admin View)
- 별도 페이지 이동 없이 앱 내부에서 구글 시트 데이터를 직접 수정/추가/삭제
- **저장하기**: 수정된 데이터를 Google Apps Script(GAS)를 통해 원본 시트에 실시간 동기화

---

## 3. UI/UX 디자인 가이드

### 3.1 컬러 시스템
- **Main Blue**: `#2563eb` (Blue-600) - 핵심 버튼 및 강조색
- **Background**: `#f0f9ff` (Sky-50) - 설원 느낌의 연한 배경
- **Accent**: 운영진(Amber), 본캐(Blue), 부캐(Slate) 각 등급별 태그 컬러

### 3.2 컴포넌트 스타일
- **Border Radius**: `2rem` (32px) 이상의 둥근 모서리로 현대적이고 부드러운 느낌 강조
- **Backdrop Blur**: `backdrop-blur-2xl`을 사용하여 유리 질감(Glassmorphism) 구현

---

## 4. 추후 기능 확장 로드맵 (Update Plan)

### 4.1 통계 기능 추가 (1단계)
- **출석률 트래킹**: 각 시간대별 참여 횟수를 누적하여 주간/월간 참여도 상위 유저 자동 선별
- **미참여자 명단**: 마스터 시트 대비 당일 미참여자 명단 별도 추출 기능

### 4.2 멀티 유저 동기화 (2단계)
- **실시간 공유**: 현재 내가 작성 중인 명단을 다른 운영진이 동시에 볼 수 있는 기능 (Firestore 등 연동)
- **동시 편집 방지**: 현재 명단을 작성 중인 운영진 표시 기능

### 4.3 이미지 필터링 강화 (3단계)
- **연맹 마크 인식**: 이미지 내 연맹 마크가 없는 타 연맹원 자동 필터링 기능
- **해상도 보정**: 저화질 스크린샷에서 닉네임 인식률을 높이기 위한 이미지 전처리 필터 적용

---

## 5. 유지보수 및 설정 안내
- **API 키 관리**: `CONFIG.API_KEY`에 주기적으로 Gemini API 키를 갱신해야 합니다.
- **GAS 엔드포인트**: 구글 시트 구조 변경 시 GAS(Google Apps Script) 코드의 컬럼 매핑 정보를 수정해야 합니다.
- **푸터 문구**: `index.jsx` 최하단 footer 섹션에서 연맹 정보를 수정할 수 있습니다.
