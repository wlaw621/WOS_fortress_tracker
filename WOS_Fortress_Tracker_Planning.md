# WOS 요새전 참여 관리 시스템 기획안 (최종본)

## 1. 프로젝트 목적
"Whiteout Survival" 요새전 및 성 채우기 참여 인원을 스크린샷 OCR 분석을 통해 자동 기록하고, 별도의 서버 비용 없이 구글 시트를 활용하여 실시간으로 관리합니다.

## 2. 시스템 구성도
1.  **사용자**: 스크린샷 업로드 및 명단 관리 (React Web)
2.  **분석기**: Gemini 2.5 Flash API (닉네임 추출 및 정제)
3.  **저장소**: Google Sheets (데이터베이스 역할)
4.  **중계기**: Google Apps Script (API 엔드포인트)

## 3. 핵심 가치 (Core Value)
- **신속성**: 수작업으로 입력하던 명단을 3초 내에 자동 추출.
- **정확성**: AI가 연맹 태그를 제거하고 마스터 데이터와 매칭하여 등급/분류 자동 할당.
- **영구성**: 구글 인프라를 활용하여 서버 유지비용 Zero 실현.

## 4. 구현된 주요 기능
- [x] **지능형 OCR**: 복잡한 게임 화면에서 닉네임만 정확히 추출.
- [x] **실시간 동기화**: 웹과 시트 간의 지연 없는 데이터 연동.
- [x] **관리자 모드**: 앱 내에서 즉시 멤버 정보를 수정할 수 있는 편의성.
- [x] **멀티 세션**: 요새전 시간대별(12, 18, 21시) 독립 데이터 관리.
